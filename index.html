Süleyman Demirel Üniversitesine dikey geçişle Isparta Uygulamalı Bilimler Üniversitesi Bilgisayar Programcılığı Bölümü'nden geldim.

Web tasarımı ve daha çok oyun geliştirme üzerine uygulamalar geliştiriyorum. Web tasarımına üniversitenin iş yeri eğitimi programı kapsamında ilgi duymaya başladım. İlk projem bir belediyenin web sayfasını yapmakla başladı daha sonra tarım ilaçları satışı yapacak olan birisinin web sitesini yapmak oldu. Github repositorysinde de bazı deneme projelerim mevcut.

Oyun projelerim ise mobil tarafında eğitim üzerineydi ancak son zamanlarda bir çalışmam olmadı.
